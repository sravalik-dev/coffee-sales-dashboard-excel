# ☕ Coffee Sales Dashboard

## Overview
Interactive sales dashboard built in Microsoft Excel to analyze coffee sales trends, customer behavior, and country-wise performance.

## Features
- Dynamic KPI Cards
- Interactive Slicers
- Timeline Filtering
- Sales Trend Analysis
- Top Customers Analysis
- Country-wise Revenue Breakdown

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- GETPIVOTDATA
- Dashboard Design
