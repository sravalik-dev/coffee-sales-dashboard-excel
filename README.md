☕ Coffee Sales Dashboard

## Overview
An interactive sales dashboard built in Microsoft Excel to analyze coffee sales trends, customer behavior, and country-wise performance. The dashboard enables dynamic filtering and visualization of key business metrics to support data-driven decision-making.

## Features
- **Dynamic KPI Cards** – Real-time summary metrics (Total Revenue, Total Orders, Avg Order Value, etc.)
- **Interactive Slicers** – Filter data by country, roast type, loyalty status, and more
- **Timeline Filtering** – Analyze trends across custom date ranges
- **Sales Trend Analysis** – Visualize revenue and order patterns over time
- **Top Customers Analysis** – Identify highest-value customers by spend
- **Country-wise Revenue Breakdown** – Compare performance across regions

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- GETPIVOTDATA
- Dashboard Design (conditional formatting, custom KPI cards)

## File Structure
- `coffee_sales_data.xlsx` – Raw sales dataset
- `Dashboard` sheet – Final interactive dashboard
- `PivotTables` sheet – Supporting pivot tables and calculations

## How to Use
1. Download/clone the repository
2. Open `coffee_sales_dashboard.xlsx` in Microsoft Excel
3. Use the slicers and timeline filter on the Dashboard sheet to explore the data interactively
